# Knox Social Command Centre — AI Marketing Challenge

## Department 2: Marketing & Branding | Week 3 Task

### The Brief

You are the Marketing Lead for your Knox AI Company. Last week you built a **Brand Book** — the rulebook for how Knox looks, sounds, and feels. This week, you **USE** that brand book to create a **Social Command Centre** — a marketing hub that generates on-brand social media content.

Everything you create must follow YOUR brand book. Consistency is what makes a brand real.

---

### Required Challenges (complete in order):

1. **Post Creator UI** — Build a social media post creator with platform selector tabs (Instagram, Twitter, LinkedIn, TikTok), a text area for the brief, and a preview panel. Use YOUR brand book colours.

2. **AI Copy Generation** — Add a Generate button that creates marketing copy from a brief. AI outputs a caption + hashtags optimised for the selected platform.

3. **Brand Voice Selector** — Add tone buttons: Professional, Casual, Hype, Gen Z. Each regenerates the caption in that tone — matching your brand book's voice guidelines.

4. **Platform Preview** — Make the preview panel update per platform. Instagram = square post, Twitter = tweet card, LinkedIn = professional card, TikTok = vertical video card.

### Experienced Challenges (pick 2+):

5. **Landing Page Builder** — Second tab that builds a product landing page (hero, features, CTA)
6. **Image Prompt Generator** — Generate AI image prompts (DALL-E/Midjourney) for each post
7. **Campaign Calendar** — Weekly calendar view with drag-and-drop scheduling
8. **Multi-Platform Export** — Same post optimised for all platforms simultaneously
9. **Analytics Dashboard** — Mock engagement metrics with charts
10. **A/B Testing Mode** — Two versions side-by-side, vote on the winner

---

### How to Build It

- Open this repo in **GitHub Codespaces**
- Check out **PROMPTS.md** for ready-to-use AI prompts for each challenge
- Use AI tools to help you code — prompt Claude, Copilot, or ChatGPT
- Reference **YOUR brand book** from last week for all design decisions
- Start with `index.html`, `style.css`, and `script.js`

### Marking Criteria

| Criteria | Weight |

### Key Rules

- Every colour, font, and tone must match your brand book
- Use Knox Blue (#1A5FAF) and dark navy (#0A0F1E) as your primary palette
- The product you're marketing is **Knox Stock Picker** from Week 1
- Prompt AI tools to help — but understand what the code does
- Commit and push regularly!
